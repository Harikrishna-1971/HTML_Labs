# HTML Labs

A comprehensive collection of HTML and CSS learning exercises and projects showcasing modern web development techniques.

## Project Overview

HTML Labs is an educational repository containing 10 different experiments, each focusing on various aspects of HTML markup, CSS styling, and web design patterns. This project is designed for learners to explore and understand the fundamentals of web development.

## Experiments

### Exp 1: Online Portfolio
**Files:** `index1.html`, `index1_1.html`, `style1.css`, `style1_1.css`

A personal portfolio website showcasing:
- Semantic HTML structure
- Header and hero section with profile image
- About section with multi-paragraph content
- Links to social media profiles (GitHub, LinkedIn, Portfolio)
- Professional footer with copyright information
- Responsive design principles

### Exp 2: Product Gallery
**Files:** `index2.html`, `style2.css`

An e-commerce product gallery featuring:
- Product card layout with images
- Product information (name, price, description)
- Grid-based responsive gallery
- Multiple product items (6 accessories)
- Clean and organized product display

### Exp 3: Workshop Registration Form
**Files:** `index3.html`, `style3.css`

An interactive form demonstrating:
- Text input fields
- Email input validation
- Select dropdown menus
- Radio button groups
- Checkbox selections
- Form submission button
- Proper form structure and labeling
